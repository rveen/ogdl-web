# Change list

## Changes to the OGDL Specification

2018-02-25

* Removed flow-type syntax: parenthesis and commas.

2017-10-06

* Some rules (EBNF) corrected
* Graph syntax corrected

2015-2016

* Level 2 production: ":" path
* Parenthesis (flow) based syntax removed 

2014-01-20

* Metainfo key removed
* Level 2 production: #=relative_path
* Productions more accurate

2011-09-20

* Level 2 simplified: #{N)
* Deleted '--' as break.
* Production 1 simplified (asume valid chars)
* Short form for tables not included because  it ads confusion in case of cycles

2005-04-03

* Some descriptive text added.
* Defined a new EOS sequence consisting of two dashes.
* Make optional the spaces around '(', ')' and ','
          
2004-06-14

* Tabs and spaces can not be intermixed in indentation.

2004-03-05

* Comments, meta-information added. 
* Semicolons deleted, comma chages meaning.
* Round-tripping chapter added.
* Some productions commented.

2003-11-17

* New cycle productions (were &amp;{} and *{}). 
* Unicode BOM mandates Unicode stream.
* Implementor decides whether he/she needs level 2. 

2003-09-02  

* Initial release
